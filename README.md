# ZipToRar

A simple Python GUI tool to convert files into .rar archives using WinRAR.

---

## Features

* Easy-to-use graphical interface (Tkinter)
* Select any file and compress it into .rar
* Choose custom output location
* Lightweight and beginner-friendly
* Uses WinRAR for compression

---

## Requirements

* Python 3.x
* WinRAR installed on your system
* WinRAR added to system PATH (rar command accessible in terminal)

---

## Installation

1. Clone the repository:
   git clone https://github.com/recker895/ZipToRar.git

2. Navigate to the project folder:
   cd ZipToRar

3. Run the application:
   python main.py

---

## Usage

1. Click "Browse" to select a file
2. Choose the output .rar file location
3. Click "Convert to RAR"
4. The archive will be created at the selected location

---

## Notes

* This tool depends on WinRAR. Make sure rar.exe is available in your system PATH.
* If the conversion fails, check your WinRAR installation.
